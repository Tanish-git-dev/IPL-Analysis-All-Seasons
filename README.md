# IPL Data Analysis

## Overview
This project analyzes historical IPL match and ball-by-ball data using Python to uncover trends in team performance, batting statistics, match outcomes, and scoring patterns.
## Dataset
The analysis uses IPL match and delivery datasets containing:
* Match details
* Team performances
* Player statistics
* Toss results
* Venue information
* Ball-by-ball records
## Analysis Performed
* Top run-scorers in IPL history
* Dismissal patterns of leading batsmen
* Impact of toss decisions on match outcomes
* Average runs scored per season
* Trends in IPL scoring over time
* Team performances across major venues
* Death-over scoring analysis for top teams
## Tools Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook
## Key Findings
The analysis highlights batting dominance, the influence of toss decisions, changing scoring trends across seasons, venue-based performance variations, and strategies adopted by successful IPL teams.
